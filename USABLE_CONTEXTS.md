### **Simple Web Interface**
```bash

python mushroom_app.py
# Open: http://localhost:5000
```

**Features:**
- User-friendly form interface
- Real-time predictions
- Confidence scores
- Safety warnings

### **API Service**
```bash
# Test the API
curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \

```
